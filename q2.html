Introduction to the World Wide Web

The World Wide Web, invented by Tim Berners-Lee, revolutionized the way we share and access information on the internet.

Key Components of the Invention

  • HTML (Hypertext Markup Language): The markup language used to structure content on the web.
  • HTTP (Hypertext Transfer Protocol): The protocol for transferring hypertext requests and information on the web.
  • URL (Uniform Resource Locator): A web address that uniquely identifies a resource on the internet.
